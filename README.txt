Tile Estimator V1 - Beta Local

Thank you for testing this app.

How to open the app:

1. Extract the ZIP file first.
2. Open the extracted folder.
3. Double click index.html.
4. Keep the css and js folders in the same folder as index.html.

Important:

- This beta saves projects on the browser/device you are using.
- It does not save projects to the cloud yet.
- Do not clear browser data unless you already exported your project backup.
- After any important estimate, click Export Project JSON and keep that file as a backup.
- To move a project to another device, use Export Project JSON and Import Project JSON.

What to test:

- Create a project.
- Add areas such as shower, floor, backsplash, fireplace, or stairs.
- Save and load the project.
- Review Materials Summary.
- Print or save Quote as PDF.
- Print or save Invoice as PDF.
- Add Evidence Log items and print or save Evidence Report as PDF.

If something goes wrong:

- Take a screenshot.
- Write down what device and browser you used.
- Write down the steps that caused the issue.
- Send the screenshot and notes to the person who shared this beta with you.

Known limitation:

This is a local beta. Projects are stored on your device/browser until cloud accounts and server backup are added later.
